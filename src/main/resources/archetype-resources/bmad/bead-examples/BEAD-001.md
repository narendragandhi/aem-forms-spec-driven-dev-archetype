# BEAD Issue Example: Create a New Form Component

- **ID:** BEAD-001
- **Title:** Create a 'First Name' Text Field Component
- **Assigned Agent:** `dev-agent`
- **Status:** To Do

## Description

Based on the `sample-form.json` spec, create a new AEM component and a corresponding React component for the 'firstName' field.

## Acceptance Criteria

1.  An AEM component is created at `/apps/<appName>/components/form/firstname`.
2.  A React component `FirstNameField.js` is created in `/ui.frontend.react.forms.af`.
3.  The component should render an HTML text input.
4.  The component's label should be "First Name".
5.  The component should be registered with the AEM Forms editor.
